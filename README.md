# dotfiles

## Install
```bash
Remove config:
  rm -rf ~/.config/nvim
  rm -rf ~/.local/share/nvim
Clone nvchad:
  git clone https://github.com/NvChad/NvChad ~/.config/nvim --depth 1
Remove custom config:
  rm -rf ~/.config/nvim/lua/custom
Clone custom config:
  git clone git@github.com:nguyen-quang-phu/dotfiles.git ~/.config/nvim/lua/custom

```
